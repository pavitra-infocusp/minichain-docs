<Note>
  Minichain is an educational blockchain implementation designed for learning. It features a register-based VM, custom assembly language, and Proof of Authority consensus.
</Note>

## What is Minichain?

Minichain is a complete, single-node, account-based blockchain built from scratch in Rust. It's designed as a learning project to understand blockchain internals by implementing every component yourself.

### Key Features

<CardGroup cols={2}>
  <Card title="Register-based VM" icon="microchip">
    16 registers with 40+ opcodes and gas metering
  </Card>
  <Card title="Custom Assembly" icon="code">
    Human-readable assembly language that compiles to bytecode
  </Card>
  <Card title="Smart Contracts" icon="file-contract">
    Deploy and execute bytecode contracts with storage
  </Card>
  <Card title="Proof of Authority" icon="shield-check">
    Round-robin block production with Ed25519 signatures
  </Card>
</CardGroup>
